# IlluminAI

The **IlluminAI Project** is developed within the Italian National PhD in Heritage Science (HS), curriculum 9 "Digital transition for Heritage", and with the support of the CNR-ISPC.

## 🔎 Main features
IlluminAI is a **WebXR application** for exploring late medieval illuminated manuscripts in a 3D space. 
The users will be able to analyze pages searching for figurative miniatures and compare these with other artworks based on their iconography. 

## 🚀 Getting started
1.	Follow **ATON Framework** instructions [https://github.com/phoenixbf/aton] for deployment;
2.	Just place (or git clone) IlluminAI app folder in /<your-ATON-root-folder>/wapps/ thus obtaining: /<your-ATON-root-folder>/wapps/illuminai-main/ ;
3.	Open http://localhost:8080/a/illuminai/ ;
4.	Upload your csv and update the app.
